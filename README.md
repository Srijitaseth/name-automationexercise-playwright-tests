# Automation Exercise Playwright Tests

This repository contains automated end-to-end test cases for [Automation Exercise](https://automationexercise.com/test_cases) using Playwright and TypeScript.

## Assignment Scope

The top 5 Automation Exercise test cases are automated:

1. Register User
2. Login User with correct email and password
3. Login User with incorrect email and password
4. Logout User
5. Register User with existing email

## Tech Stack

- Playwright
- TypeScript
- Node.js
- Faker.js
